Как работает АвтоБот?

Всё очень просто. Программа располагает деревом "Ботов", которые создаются и программируются Вами. Вам просто нужно добавить нового "бота" в ветку дерева "Мой компьютер". Переключиться на "бота" и во вкладке "Схема" составить из компонентов необходимую последовательность действий, которые будут выполняться по событию настроенного Вами "триггера". Далее Вы жмете кнопку "Старт". И Ваш "бот" начнёт работу. После чего Вам достаточно свернуть окно программы, и она займёт своё место в Tray-зоне рядом с часами, выполняя рутинную работу ботов.

Характеристики

АвтоБот старается потреблять минимум ресурсов памяти и процессора. Разработчики АвтоБота постоянно заботятся об оптимизации скорости выполнения команд программы, ведут борьбу за каждый байт в памяти. Но самой главной целью является безотказность и стабильность работы приложения.

АвтоБот экономно использует оперативную память. В память загружаются только те боты, которые вы настраиваете, либо те которые выполняются в данный момент. Все остальные боты, спокойно хранятся в бд стараясь не занимать лишнего места в оперативной памяти.

Для оптимизации скорости выполнения, алгоритм схемы компилируются в скрипт код, который в свою очередь перед выполнением компилируется в байт код.

АвтоБот включает следующие возможности:

  • Позволяет запрограммировать макропоследовательность действий и запускать их на выполнение когда будет нужно.
  • Программировать реакцию на определённое системное событие.
  • Запускаться сразу после старта ОС.
  • Включает компонент: "Триггер" для настройки срабатывания бота.
  • Включает компонент: "Диалоговые окна" для вызова диалоговых окон ОС.
  • Включает компонент: "Работа с окнами" для различного рода манипуляций над окнами ОС.
  • Включает компонент: "Работа с процессами" для манипуляций над процессами в ОС, либо запуска сторонних приложений.
  • Включает компонент: "Работа с файлами" для различных манипуляций с файлами.
  • Включает компонент: "Работа с почтой" для операций с почтой. Получение почты, отсылка, поиск в ящике, удаление из ящика.
  • Включает компонент: "Скрипт" для работы со скриптами. Поддерживается скриптовый язык SQUIRREL.
  • Включает инструмент "Шпион" для поиска и извлечения информации об открытых окнах ОС.
